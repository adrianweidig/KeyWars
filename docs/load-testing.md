# Lasttests

Das Tool `tools/KeyWars.LoadTest` simuliert lokale Raumteilnehmer ohne externe Dienste. Es misst Progress-P95 und Zieleinläufe.

Für belastbare Betreiberwerte muss der Test auf der Zielhardware mit den geplanten Grenzen laufen. Ergebnisse gehören als Release- oder Betriebsartefakt abgelegt.
