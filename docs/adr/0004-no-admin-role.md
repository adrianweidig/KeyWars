# ADR 0004: Keine Administratorrolle

## Kontext

Die Anwendung soll keine privilegierte Oberfläche besitzen.

## Entscheidung

Alle Nutzer haben dieselben Anwendungsrechte. Verwaltung erfolgt über Eigentümerschaft.

## Konsequenzen

Keine Adminnavigation, keine lokale Freischaltung und keine globale Moderation.
