# Benutzerhilfe

Nach der Anmeldung zeigt die Startseite Sofortaktionen, Missionen, letzte Kennzahlen, Serie, Rating und Empfehlung.

Bereiche:

- **Spielen**: Sprint, Wörtertest, Texttraining, Fehlerfokus und Geisterrennen.
- **Arena**: Live-Räume starten oder per Code beitreten.
- **Texte**: eigene Texte, Uploads und Sammlungen.
- **Profil**: Ziele, Erfolge, Einstellungen, Selbstexport und Löschung.

Personensuche findet nur Menschen, die sich mindestens einmal in KeyWars angemeldet haben.
