# Code of Conduct

## Our Pledge

We are committed to a respectful, professional and harassment-free project environment.

## Expected Behavior

- Be direct, factual and constructive.
- Focus criticism on code, design and operational impact.
- Assume good intent while still requiring clear evidence for technical claims.

## Unacceptable Behavior

- Harassment, threats or discriminatory language.
- Publishing private information without permission.
- Repeated off-topic disruption of project work.

## Enforcement

Project maintainers may remove comments, close issues or restrict participation when behavior conflicts with this code of conduct.
